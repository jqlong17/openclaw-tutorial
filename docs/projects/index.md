# 实践项目

从入门到企业级的完整项目实战。

## 项目分级

### 入门级（第21章）
适合初学者的简单项目，快速上手 OpenClaw 核心功能。

- **智能客服机器人** - 多平台接入 + 知识库问答
- **个人知识助手** - 记忆系统 + 定时任务

### 进阶级（第22章）
结合外部 API 和复杂业务逻辑的项目。

- **智能家居控制中心** - Home Assistant 集成 + 场景自动化
- **AI内容创作助手** - 热点抓取 + 内容生成 + 多平台发布

### 高级（第23章）
多模态处理和复杂工作流的项目。

- **智能会议纪要助手** - 语音转录 + AI总结 + 多平台分发
- **多模态智能客服** - 图文视频故障诊断

### 企业级（第24章）
综合所有能力的完整解决方案。

- **企业智能中台** - 统一接入 + 智能路由 + 企业系统集成
