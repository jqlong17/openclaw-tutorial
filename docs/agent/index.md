# AI Agent

深入理解 OpenClaw 的 AI Agent 核心能力。

## 核心模块

### Agent 运行器
管理 Agent 的生命周期，处理消息循环，维护对话上下文。

### 工具系统
扩展 Agent 能力的机制，让 AI 能够调用外部功能。

### 记忆系统
基于 RAG 的长期记忆，支持向量检索和全文搜索。

### 媒体理解
图像识别、语音处理、视频分析的多模态能力。

## 架构概览

```
用户消息
    ↓
Agent Runner
    ↓
工具选择 → 工具执行 → 结果返回
    ↓
记忆检索 → 上下文构建
    ↓
LLM 生成回复
    ↓
发送给用户
```
