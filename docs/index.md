## 教程概览

本教程共 **24 章**，分为 **6 大模块**，带你从入门到精通 OpenClaw：

| 模块 | 章节 | 内容 |
|------|------|------|
| **基础入门** | 1-3章 | OpenClaw 概览、环境搭建、第一个 Agent |
| **核心概念** | 4-6章 | 消息传输流程、网关架构、通道抽象层 |
| **平台集成** | 7-11章 | Discord、Telegram、飞书、iMessage 深度集成 |
| **AI Agent** | 12-15章 | 运行器、工具系统、记忆系统、媒体理解 |
| **高级特性** | 16-20章 | 定时任务、插件系统、多节点、安全、性能 |
| **实践项目** | 21-24章 | 从入门到企业级的完整项目实战 |

## 快速开始

```bash
# 安装 OpenClaw CLI
npm install -g @openclaw/cli

# 创建新项目
openclaw init my-agent

# 启动开发服务器
openclaw dev
```
